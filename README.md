# Movies app

This is a React application, that allows user to search for movies by name and
find out more information about the film.

The application contains 3 pages:

- Home page - the gallery with popular movies.
- Search movies page - the page where user can search for movies by name;
- Movie page - the page where user can find out detailed information abot the
  film, its cast and reviews.

## Technology stack

- React
- React Router
- Axios
- Emotion styled

## Getting Started

To get started with this project, follow these steps:

1. Clone this repository to your local machine.
2. Navigate to the project directory.
3. Install the required dependencies using

```bash
   npm install
```

or

```bash
  yarn install
```

4. Start the development server with

```bash
   npm start
```

5. Open your web browser and visit
   [http://localhost:3000/](http://localhost:3000/) to access the application.
